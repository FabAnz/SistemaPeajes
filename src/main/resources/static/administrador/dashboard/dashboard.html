Panel administrador

Emular Tránsito

Simula el paso de un vehículo por un puesto de peaje para realizar pruebas y verificar el funcionamiento correcto del sistema de cobro automático de tarifas.

Asignar Bonificaciones

Asigna bonificaciones especiales a propietarios específicos para que puedan beneficiarse de descuentos en determinados puestos de peaje del sistema.

Cambiar Estado de Propietario

Modifica el estado de un propietario en el sistema, permitiendo cambiar entre estados como habilitado, suspendido, penalizado o deshabilitado según sea necesario.
